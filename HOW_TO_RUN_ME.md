# How to Run Me

This project is a Spring Boot application.

## Repository

GitHub: https://github.com/rollins2131/Gaming-tournament-platform.git

## Project Folder

`C:\Users\HP\Documents\Project\Project`

## Run Option 1: VS Code Task (recommended)

1. Open this folder in VS Code.
2. Press `Ctrl + Shift + P`.
3. Type: `Tasks: Run Task`.
4. Select: `Run Spring Boot App`.

## Run Option 2: Terminal Command

1. Open a terminal in the project folder.
2. Run:

```powershell
& .\mvnw.cmd clean spring-boot:run
```

## App URL

After startup completes, open:

`http://localhost:8080/login`

## Stop the App

In the same terminal where it is running, press:

`Ctrl + C`

If prompted for termination confirmation, press `Y` and Enter.
